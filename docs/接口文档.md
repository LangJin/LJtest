# LJtest接口文档
## 前台接口
### 获取首页轮播图
- 接口地址：/get_title_img?num=1
- 接口类型：get
- 请求头：
```json
{"Content-Type":"application/json"}
```
- 接口参数：
```json
无
```
- 返回样式
```json
[
    {
        "content": "人啊就是要努力",
        "id": 1,
        "imghost": "/images/1.jpg",
        "title": "不积跬步无以至千里"
    }
]
```
